导航零号

免费的导航软件，有 iOS 和 Android 两个版本。苹果 AppStore，华为商店，小米商店均可下载。

具备搜索、算路、路况等全部功能。它是个样板项目，体现了导航引擎最新的功能。 为各种定制导航项目趟平了产品化的道路。

并且，它已经内部全部开源。各项目部可以在它的基础上做改造，做项目定制。

build 导航SDK

支持主流平台 iOS、Android、嵌入式Linux。 使用它，可以轻松的开发出类似导航零号的软件。

iNaviCore(iOS SDK) jNaviCore(Android SDK)已经全部内部开源。 已经成为智联NGLP的基础，并且在企业服务中广泛应用。

cast_connected API

包括在线算路、搜索、路况信息等全套的在线API。 它是在线导航的基础，导航SDK依赖于它完成在线功能。

我们提供直接调用，也支持本地部署。其中算路服务因为定制需求比较多，也已经开源，供各项目部定制开发。

开源项目

我们的C++服务器架构、地图数据切割算法等已经开源。

导航零号

导航零号是引擎部直接开发的、面向大众的导航软件。 它有 iOS 和 Android 两个版本。在苹果 AppStore，华为应用市场，小米应用商店均可下载。

导航零号具备搜索、算路、交通路况等全部功能。 它是个样板项目，体现了导航引擎最新的功能。 为各种定制导航项目趟平了产品化的道路。

导航SDK

支持主流平台 iOS、Android、嵌入式Linux。 使用它，可以轻松的开发出类似导航零号的软件。

iNaviCore(iOS SDK) jNaviCore(Android SDK)已经全部内部开源。 已经成为智联NGLP的基础，并且在企业服务中广泛应用。

API

包括在线算路、搜索、路况信息等全套的在线API。 它是在线导航的基础，导航SDK依赖于它完成在线功能。

我们提供直接调用，也支持本地部署。其中算路服务因为定制需求比较多，也已经开源，供各项目部定制开发。

ITE

导航功能集成测试环境。展示了导航引擎全部前沿功能。 它是和引擎同步开发的，主要用于引擎的集成开发。测试人员也用它测试引擎功能。 产品人员用它编辑地图配色。

开源项目

我们的C++服务器架构、地图数据切割算法等已经开源。

文档

引擎的公开文档、开发者指南、竞品分析等。
